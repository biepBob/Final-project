Examentaak programmeren

Academiejaar 2017-2018

In de map vindt u een pdf file van het verslag. Verder vind u een drie mappen. 
De eerste van deze mappen bevat de reeds getrainde netwerken als csv files die 
kunnen worden geladen. De tweede map bevat alle code en is verder opgedeeld in 
6 mappen, 1 voor elke groep en 1 voor de propagateinterface klasse.  De mappen 
voor groep 1, 2, 5 en propagateinterface bevatten simpelweg code voor de rele-
vante klassen.  De mappen voor groep 3 en 4 bevatten respectievelijk de volle-
dige code voor  RooT en de Gui  met makefiles erbij.  Tenslotte bevat de derde
map enkele  afbeeldingen  uit de mnist database in jpg  formaat om gebruikt te
worden bij het inladen van individuele afbeeldingen.
